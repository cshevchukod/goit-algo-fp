# Завдання 7 — Метод Монте-Карло (два кубики)

## Опис
Програма імітує велику кількість кидків двох гральних кубиків.  
Для кожного кидка обчислюється сума (2–12), підраховується кількість появ кожної суми та визначається її ймовірність.

## Аналітичні ймовірності
Для двох чесних кубиків існує 36 рівноймовірних комбінацій. Ймовірності сум:

- 2  → 1/36 = 2.78%
- 3  → 2/36 = 5.56%
- 4  → 3/36 = 8.33%
- 5  → 4/36 = 11.11%
- 6  → 5/36 = 13.89%
- 7  → 6/36 = 16.67%
- 8  → 5/36 = 13.89%
- 9  → 4/36 = 11.11%
- 10 → 3/36 = 8.33%
- 11 → 2/36 = 5.56%
- 12 → 1/36 = 2.78%

## Результати Монте-Карло (приклад)
При кількості кидків `100000` отримані ймовірності близькі до теоретичних.
Наприклад:
- сума **7** виходить найбільш імовірною (близько **16.67%**),
- суми **2** та **12** — найрідші (близько **2.78%**).

Невеликі відхилення від аналітичних значень є нормальними, бо метод Монте-Карло використовує випадкову симуляцію.

## Висновок
Отримані методом Монте-Карло результати узгоджуються з аналітичними розрахунками.  
Якщо збільшувати кількість кидків (`trials`), то ймовірності ще ближче наближаються до теоретичних значень.
